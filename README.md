# Habit Tracker Backend (Node.js + Express)

A backend API service for a Habit Tracker application, providing CRUD operations for managing user habits. Includes a minimal static UI for testing and rapid prototyping.

## Features
- RESTful CRUD API for habits
- JSON file-based local persistence (`data/habits.json`)
- Lightweight Express architecture
- Simple static UI for debugging
- Easy to extend for databases (MongoDB, PostgreSQL)
- Ideal for learning + demos + hackathons

## Tech Stack
- Node.js
- Express.js
- Nodemon (development)
- JSON Persistence
